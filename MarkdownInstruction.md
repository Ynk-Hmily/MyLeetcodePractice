# Markdown 基本语法速查表

| 目的 | 语法示例 | 说明 |
|------|-----------|------|
| 一级标题 | `# 一级标题` | 使用 `#`，数量表示层级（1~6） |
| 二级标题 | `## 二级标题` | `##` 表示二级标题 |
| 三级标题 | `### 三级标题` | 类推，最多 `######` |
| 粗体 | `**粗体**` | 两个 `**` 包裹文字 |
| 斜体 | `*斜体*` | 一个 `*` 包裹文字 |
| 粗斜体 | `***粗斜体***` | 三个 `*` 包裹文字 |
| 无序列表 | `- 项目` 或 `* 项目` | 可以嵌套，用空格缩进 |
| 有序列表 | `1. 项目` | 数字加点，自动编号 |
| 引用 | `> 引用内容` | 用于引用文本 |
| 内联代码 | `` `代码` `` | 单行代码片段 |
| 代码块 | <pre>```python<br>print("hello")<br>```</pre> | 三个反引号 + 可选语言，高亮显示 |
| 水平分割线 | `---` 或 `***` | 创建水平线分隔内容 |
| 链接 | `[显示文字](https://example.com)` | 点击文字跳转链接 |
| 图片 | `![alt](图片链接)` | 显示图片，alt 为替代文本 |
| 表格 | <pre>| 列1 | 列2 |<br>|---|---|<br>| 内容1 | 内容2 |</pre> | 使用 `|` 和 `-` 来创建表格 |
| 任务列表 | `- [ ] 待办` <br>`- [x] 完成` | 可以做打勾列表 |
| 公式（需支持插件） | `$公式$` | 行内公式 |
| 公式块 | `$$公式$$` | 独立公式块 |
| 注脚 | `[^1]` + 底部 `[^1]: 注释内容` | 文档注释或补充说明 |

---

💡 **使用建议：**
- 用标题分级组织笔记内容
- 代码块和公式块配合插件（如 Markdown Preview Enhanced）显示最佳
- 表格和列表可用于整理算法复杂度、测试用例等信息
